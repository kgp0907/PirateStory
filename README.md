PirateStory
Unity로 제작된 수집형 방치형 RPG 포트폴리오입니다.

회사에 들어가 첫 메인 프로그래머를 맡은 게임입니다.

모든 UI, 콘텐츠, 게임시스템을 구현했습니다.

현재 Git 에는 프로젝트의 일부 샘플코드만 있습니다.
